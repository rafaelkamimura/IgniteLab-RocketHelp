# IgniteLab-RocketHelp
## Inicialização:
```bash
git clone https://github.com/rafaelkamimura/IgniteLab-RocketHelp.git
cd IgniteLab-RocketHelp
yarn add global expo-cli
yarn add native-base
yarn install
expo install


```
